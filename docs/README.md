# Documentation

Bem-vindo à documentação do **Rtrack**, uma plataforma de acompanhamento de serviços jurídicos que permite que advogados monitorem o progresso dos processos, forneçam pareceres ao cliente via dashboard, acompanhem etapas financeiras, recebam recomendações de IA, acessem vídeos (rcast) e conteúdo de blog.
